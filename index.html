About Me

My name is Sam Foster and I'm a 15 year old developer and designer and I've been "programming" for the past 6 years! I started out with Scratch, a block coding platform, and eventually moved on to high-level languages like Lua and Python. I stuck with those for a while, but a few years ago, I learned Javascript to aid in web development. Today, I prefer lower-level languages like Rust and C. My dream is to eventually create an operating system from scratch, but I've got a lot to learn before then!
